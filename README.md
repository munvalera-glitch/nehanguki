## 🚀 Как отправить обновления на сервер?

В корне проекта есть специальный скрипт `deploy.sh`. Как только вы (или AI-ассистент) закончили редактировать код, просто выполните команду в терминале:

```bash
./deploy.sh
```

Этот скрипт:
- Скопирует только измененные файлы на сервер (без учета тяжелой папки `node_modules`).
- Установит новые зависимости на сервере.
- Соберет фронтенд.
- Автоматически перезапустит бота через `pm2`.
